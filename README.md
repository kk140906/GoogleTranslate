基于Qt5的谷歌翻译，鼠标点击非翻译软件区域时软件自动隐藏。小软件，自己用，懒得美化!
中英文自动识别，可在选定需要翻译的内容后按F6一键翻译，也可在输入框输入内容进行翻译，在输入内容时为了减少谷歌翻译网页的请求次数，请求间隔时间为300ms。出现网络错误需安装Win64OpenSSL_Light-1_1_1d
